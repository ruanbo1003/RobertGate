# 相册 PRD

## 概述
公开的照片相册页面，以瀑布流形式展示服务器指定目录下的所有照片，支持点击查看大图、左右翻页和全屏浏览。

## 目标用户
- 网站访客（无需登录）
- 核心诉求：浏览照片，查看感兴趣的照片细节

## 使用场景
1. 访客打开相册页，看到瀑布流照片墙，滚动浏览
2. 看到感兴趣的照片，点击进入 Lightbox 查看大图，左右翻页
3. 想看细节，点全屏按钮放大浏览，ESC 退出

## 功能需求

### P0（必须实现）
- [ ] 瀑布流照片网格：3 列（桌面）/ 2 列（平板）/ 1 列（手机），横竖屏照片自适应混排
- [ ] Lightbox 查看模式：点击照片弹出大图，暗色遮罩 + 居中显示
- [ ] Lightbox 切换：左右箭头按钮 + 键盘 ← → 切换上下张，到头禁用箭头
- [ ] Lightbox 关闭：ESC 键或点击关闭按钮/遮罩返回网格
- [ ] 全屏模式：Lightbox 右上角全屏按钮，使用浏览器 Fullscreen API，ESC 退出全屏（回到 Lightbox）
- [ ] 后端 API：读取服务器写死目录（`/home/ubuntu/photos`）返回照片列表（文件名、宽高、URL）
- [ ] Nginx 静态服务：照片通过 Nginx 直接提供

### P1（优先级次之）
- [ ] 懒加载：照片滚动到可视区域才加载（IntersectionObserver）
- [ ] 缩略图：网格用小尺寸，Lightbox 加载原图
- [ ] 照片加载动画：骨架屏占位 + 加载完成渐入
- [ ] 键盘快捷键：← → 切换，F 全屏，ESC 退出
- [ ] 触屏手势：移动端左右滑动切换

## 页面布局

采用瀑布流布局（Masonry）：

```
┌─────────────────────────────────┐
│  Navbar                          │
├─────────────────────────────────┤
│  页面标题："Gallery"              │
├─────────────────────────────────┤
│  瀑布流网格                       │
│  ┌───┐ ┌─────┐ ┌───┐           │
│  │   │ │     │ │   │           │
│  │   │ └─────┘ │   │           │
│  └───┘ ┌───┐   └───┘           │
│  ┌─────┐│   │ ┌─────┐          │
│  │     ││   │ │     │          │
│  └─────┘└───┘ └─────┘          │
│  ... 懒加载更多 ...              │
├─────────────────────────────────┤
│  Footer                          │
└─────────────────────────────────┘

Lightbox 叠加层：
┌─────────────────────────────────┐
│  暗色遮罩 (bg-black/80)          │
│         ┌─── 右上角 ───┐        │
│         │ [全屏] [关闭] │        │
│  [←]    │    大图       │   [→] │
│         └──────────────┘        │
│              1 / 50              │
└─────────────────────────────────┘
```

移动端：网格变 1 列，Lightbox 箭头隐藏（用滑动手势），按钮增大。

## 交互设计

- **加载态**：网格区域显示骨架屏占位块（模拟瀑布流形状）
- **空状态**：目录无照片时显示"暂无照片"提示 + 图标
- **错误态**：API 请求失败显示错误提示 + 重试按钮
- **照片 hover**：轻微放大 (scale 1.02) + 浅色遮罩
- **Lightbox 进入**：照片点击后遮罩淡入 + 大图缩放进入
- **Lightbox 切换**：左右滑动过渡（framer-motion AnimatePresence）
- **全屏按钮**：进入全屏时图标切换为退出全屏图标
- **ESC 行为**：全屏状态下 ESC → 退出全屏（保留 Lightbox）；非全屏 Lightbox 下 ESC → 关闭 Lightbox 回网格

## 边界情况

- 第一张照片：左箭头禁用（灰色不可点击）
- 最后一张照片：右箭头禁用
- 照片加载失败：显示 broken image 占位图
- 目录不存在/权限不足：API 返回错误，前端显示友好提示
- 几百张照片：懒加载，只加载可视区域 + 缓冲区的照片

## 设计约束

沿用现有设计系统（`src/styles/design-tokens.css`）：
- 字体：Plus Jakarta Sans
- 页面背景：#F8FAFC + 网格线纹理
- 照片卡片：白色圆角 8px + shadow-sm，hover 加深为 shadow-md
- Lightbox 遮罩：bg-black/80
- 按钮：与现有 Button 组件风格一致
- 间距：照片间距 12px（gap-3）

## 技术备注

### 路由
| 路由 | 页面 |
|------|------|
| `/gallery` | GalleryPage |

### API
| 方法 | 路由 | 说明 |
|------|------|------|
| GET | `/api/v1/gallery/photos` | 返回照片列表 |

### 响应格式
```json
{
  "code": 0,
  "data": {
    "photos": [
      {
        "filename": "IMG_001.jpg",
        "url": "/photos/IMG_001.jpg",
        "thumbnail_url": "/photos/thumbs/IMG_001.jpg",
        "width": 4032,
        "height": 3024
      }
    ],
    "total": 150
  },
  "message": "ok"
}
```

### 照片目录
- 服务器路径：写死在后端代码中（`/home/ubuntu/photos`）
- Nginx 配置：`location /photos/ { alias /home/ubuntu/photos/; }`

### 缩略图（P1）
- 后端首次启动时或 API 调用时生成缩略图到 `thumbs/` 子目录
- 缩略图宽度：400px，保持比例
